# 故事：动摇的心

你接过那瓶水，喝了一口。

凉意顺着喉咙滑下去，让你混沌的脑袋清醒了一些。你不知道该怎么回答她的问题。"你真的什么都不记得了吗"——这个问题像一把钥匙，插在一扇锈迹斑斑的门上，却怎么也打不开。

"我不知道。"你说，声音沙哑得像是自己的，又像是别人的，"我醒来的时候……就什么都不记得了。"

小夏没有说话。

她只是点了点头，肩膀微微下沉，像是被什么东西压了一下。然后她抬起头，看向那棵樱花树。阳光穿过树叶的缝隙，在她的脸上投下斑驳的光影。

"那我告诉你吧。"

她说。

"我们是在大一的时候认识的。图书馆，你坐在靠窗的位置，我在你对面坐下。你抬头看了我一眼，然后笑了。"

她的嘴角微微上扬，像是在回味某个珍贵的瞬间。

"那是我见过的最好看的笑容。"

她转过头，看向你。阳光落在她的眼睛里，让那双眼睛看起来格外明亮——又或者，那只是因为她一直在忍着泪水。

"从那天起，我们就成了朋友。然后是……"

她顿了顿，像是在斟酌用词。

"……比朋友更重要的人。"

她说这话的时候，声音很轻很轻，像是在说一个不能让第三个人听到的秘密。

"四年了。"她说，"我们认识四年了。"

风吹过，樱花树发出沙沙的声响。花瓣从枝头落下，飘落在你们之间的空隙里。

"我知道你最近……经历了一些事情。"小夏的声音变得更轻了，"可能会很难熬。但是……"

她转过头，直视着你的眼睛。

"但是我想让你知道，你不是一个人。"

她的手慢慢地伸过来，小心翼翼地碰了碰你的手背。那个动作很轻，像是在试探你的反应。

"我会陪着你。"

"一直。"

---

*你就那样坐着，任由她的手指轻轻触碰着你的手背。*

*你不记得她。*

*但你的身体记得。*

*你的心跳在说谎——它跳得太快了，快得像是见到了什么人。*

*而那个人，就在你身边。*
