# 故事：小夏的眼泪

*——记忆闪回——*

"我不想忘记……"

这句话从黑暗中传来，回荡在某个你看不见的地方。

你看到了自己。蜷缩在角落里，像是一只受伤的幼兽。不知道是因为什么，你崩溃了。你在哭，在发抖，在试图抓住什么却什么都抓不住。

四年的记忆。无数的日子。那些你以为理所当然的日常。

它们都在消失。像是被什么东西吸走，一点一点地，不留痕迹。

然后小夏来了。

她什么都没问，什么都没说。只是蹲下来，把你抱进怀里。

她的怀抱很温暖。她的心跳隔着衣服传来，一下一下，稳定而有力。

"我不想忘记那些……"你在她怀里哭着说。

你知道自己在说什么。那些人，那些事，那些感情——你都不要忘记。

特别是她。

特别是小夏。

"没关系。"她说。

她的声音也在抖。但她还是在安慰你。

"就算忘记了……我也会替你记得。"

*——记忆断裂——*

这句话是承诺。也是告白。

但你不记得自己有没有回应过这个承诺。你不记得那之后发生了什么。

你只知道，当你再次"醒来"的时候，已经是毕业典礼的早晨了。

而你什么都不记得。

*除了她。*

就算忘了所有的事，你还是记得有个人在黑暗中抱着你，告诉你她会替你记得。
