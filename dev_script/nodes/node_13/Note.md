# 线索笔记

## 记忆碎片

**场景**
- 地点：某个黑暗的房间（看不清）
- 状态：你蜷缩在角落哭泣
- 情感：极度崩溃、绝望

**关键对话**
- "我不想忘记……"——你
- "我不想忘记你……"——你
- "就算忘记了，我也会替你记得。"——小夏

**动作**
- 小夏抱住了你
- 她轻轻拍着你的背
- 什么都没问

## 重大发现

**你曾经濒临崩溃，请求不要忘记。**
**小夏承诺会替你记得。**

这说明：
- 你的失忆可能不是意外
- 有人"允许"你忘记
- 小夏是知情者，她选择了守护

## 核心疑问

- 当时发生了什么，让你想要忘记？
- 是谁拿走了你的记忆？
- "梦境协议"是什么？

## 提示

小夏一直都知道。她选择让你忘记，是因为……

也许去问问她本人，会得到答案。
