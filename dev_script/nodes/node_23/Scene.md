# 场景：Sweet: 新的开始

## 地点
毕业后的某个清晨，两人同居的小公寓

## 环境

阳光从窗帘的缝隙透进来，在地板上画出一条金色的线。

你睁开眼睛，天花板上还残留着昨晚台灯的光晕。空气里有咖啡的香味，淡淡的，从客厅的方向飘过来。远处有水声——她在洗碗。

你侧过头，看到枕边还有几根长发，是她的。枕头上还留着她的气息，淡淡的洗发水味道，混合着某种让你安心的温度。

你突然想起了一切。

好的，坏的，所有的。

那些甜蜜的清晨，你们一起在食堂吃早餐。
那些安静的下午，你们在图书馆并肩看书。
那些争吵的夜晚，你们红着眼眶却谁都不肯低头。
那场暴雨，她站在雨里哭着问你为什么不肯说出口。
那个约定，在樱花树下，你们许下的永远。

还有那个被封印的记忆——那些痛苦的、你不愿面对的、你曾经拼命想要逃避的东西。

但现在，你不害怕了。

浴室的门开了。小夏披着湿漉漉的头发走出来，看到你醒了，露出一个有些惊讶的表情。

"你醒了？"她轻声说，"我还以为你会多睡一会儿。"

她走到床边，弯下腰，额头轻轻抵着你的额头。她的头发还没擦干，水珠滴在你脸上，凉凉的。

"早安，"她说，微微笑了，"今天，我们去哪里？"
