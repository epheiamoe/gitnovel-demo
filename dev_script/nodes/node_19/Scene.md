# 场景：Sweet: 握住的手

## 地点
樱花树下，夕阳西下

## 环境

阳光从云层的缝隙中倾泻而下，把整片樱花林染成了温暖的金粉色。

风轻轻吹过，带起无数花瓣。它们在空中旋转、飞舞，像是粉色的雪，像是某种无声的祝福。樱花落在你的肩上，落在你的发丝上，落在你和小夏交握的手上。

你们就站在樱花树下。

小夏今天穿着一件白色的连衣裙，外面套着学士服。风吹起她的裙摆和发丝，在逆光中像是一幅画。她看着你，眼眶红红的，却在笑。

"你确定吗？"她问，声音有些哽咽。

你点点头。

"重新开始……意味着你要放弃那些记忆。"她说，"我们曾经一起经历过的那些。你确定你愿意？"

你再次点头。

然后你开口，声音有些沙哑，却很坚定："我不需要记得过去。我只想和你在一起。"

小夏愣住了。

然后，她的眼泪终于落下来。

她没有擦，任由它们滑过脸颊，在阳光下闪闪发光。她笑了，笑得那么用力，像是把所有的委屈、所有的等待、所有的痛苦都化成了这个笑容。

"好。"她说，声音轻得像是一声叹息，"那我们重新开始。"

她伸出手。

你握住了。

她的手指微凉，却在你的掌心里慢慢温暖起来。樱花落在你们交握的手上，像是某种无声的誓言。

你们就这样站着。

谁都没有说话。

不需要说。

夕阳把天空染成了橙红色，樱花在风中纷纷扬扬。远处有鸟鸣，有笑声，有这个毕业季特有的热闹与喧嚣。但此刻，世界仿佛只剩下你们两个人。

时间像是静止了。

又像是终于开始流动。
