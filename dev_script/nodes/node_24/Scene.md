# 场景：Meta: 署名

## 地点
某个意外的时刻——你打开了一个文件

## 环境

这是一个普通的下午。

小夏出门去买东西了，留下你一个人在家。你有些无聊，坐在她的电脑前，随手翻看着文件夹。

她的电脑里有很多东西——课程PPT、旧照片、还有一些你从没注意过的小视频缩略图。你漫无目的地浏览着，直到你看到了那个文件夹。

"空白毕业典礼"。

你愣了一下。

这个名字……有些奇怪。你不确定是不是自己看错了。你揉了揉眼睛，再看一遍——

没错。

文件夹的名字就是"空白毕业典礼"。

你的心跳漏了一拍。

你点开那个文件夹。里面只有一个文件，一份Word文档，名字叫"空白毕业典礼.docx"。

你盯着那个图标看了很久，手指悬在鼠标上方，迟迟没有点下去。

一种奇怪的预感涌上心头。

最终，你还是点开了。

文档打开了。

第一行写着——

**"作者：[你的名字]"**

你的名字。

你的名字就在那里，黑色的字体，明晃晃地刺进眼睛里。

你愣住了。

这不是恶作剧。这不是巧合。你的名字，就在这份文档的第一行。

你继续往下看。第二行是标题——"空白毕业典礼"。第三行是副标题——"一个关于记忆、失去和重新开始的故事"。

你继续往下滑动。

文档里是密密麻麻的文字。你看到了熟悉的内容——毕业典礼、失忆、樱花、小夏……

你看到了自己经历过的一切。

你看到了那些选择。

你看到了BE和Sweet两个结局。

你看到——

"这个故事献给我最好的朋友。"

你的手开始发抖。
