# 场景：毕业典礼

礼堂里人声鼎沸。

你坐在人群中，周围是穿着学士服的学生和满脸欣慰的家长。空气中弥漫着花香和消毒水的气味，舞台上挂着"2026届毕业典礼"的横幅。

一切都那么正式，那么庄重。

但你的心跳得很快。手心里还残留着那份文件夹的触感——"梦境协议"四个字像烙印一样刻在你的脑海里。

然后你听到了。

"请XXX同学和小夏同学上台……"

你愣住了。

台上，主持人正在微笑着看向你们的方向。周围的人开始窃窃私语，有人用胳膊肘捅了捅你。

"去吧，叫你呢。"

小夏坐在你旁边。她今天化了淡妆，头发扎成了马尾，穿着整齐的学士服。她的眼睛看向舞台，然后又转向你。

她微微一笑。

那个笑容很温柔。也很苦涩。像是在说：我知道你会记得的。

"去吧。"她说，声音很轻，"不管你怎么选，我都在。"

她的手伸过来，在你的手背上轻轻拍了拍。

然后她站起身，向舞台走去。

人群自动为她让开一条路。她的背影在人群中穿行，学士服的袍子在灯光下微微飘动。

你看着她越走越远。
