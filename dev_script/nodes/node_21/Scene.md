# 场景：BE: 空白之后

## 地点
毕业后的公寓，一个人

## 环境

日子一天天过去。

你一个人住在这间小公寓里。房租很贵，但你付得起——父母留下的钱足够你挥霍一阵子。窗台上有一盆绿萝，是搬进来的时候买的，叶子绿油油的，长得很好。

你每天做着差不多的事。起床，上班，下班，回家，睡觉。周而复始，像是某种精密的机器在运转。你不怎么说话，也不怎么出门。同事觉得你性格内向，但也仅此而已。

没有人知道你失去了什么。

因为你也不知道。

你偶尔会想起一些碎片。

樱花。很美的樱花，粉色的，落在肩上的。
雨。很大很大的雨，打在窗户上，看不清外面的世界。
图书馆的窗户。阳光从那里照进来，空气中有一股旧书的味道。
还有一个笑容。很温柔很温柔的笑容，像是春天的风。

但你抓不住它们。

每当你想仔细去看那些画面，它们就像握紧的沙子一样从指缝间流走。你只能眼睁睁地看着它们消失，剩下手指间残留的一点点触感。

然后，连触感也消失了。

今天整理手机的时候，你翻到了一张照片。

照片里有两个人。一个是你——笑容有些僵硬，像是不知道该怎么做表情。另一个是一个女生，长头发，眼睛弯弯的，笑得很开心。她挽着你的手臂，头靠在你肩上。

你盯着那张照片看了很久。

照片的日期是两年前。

但你想不起她的名字。

---

## 可见物品
- 手机里的照片（你和那个女生的合影）
- 窗台的绿萝
- 空荡荡的房间
- 窗外灰蒙蒙的天空

## 可探索方向
- 尝试搜索那个名字（手机里也许有线索）
- 把照片删掉
- 继续这样活下去
