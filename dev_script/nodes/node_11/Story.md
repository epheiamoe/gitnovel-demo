# 故事：被遗忘的约定

*——记忆闪回——*

"永远在一起"。

你看着纸条上的四个字，心跳加速。

这不是一时冲动的玩笑。你能感觉到当时自己的心情——那是认真的，郑重的，像是在许下某种誓言。

"拉钩。"小夏伸出小指。

你犹豫了一下，然后也伸出手，勾住了她的小指。

"拉钩上吊，一百年不许变。"她笑着说，然后踮起脚尖，在你的额头上轻轻碰了一下，"这是盖章。"

你的脸一下子红了。

夕阳更红了，把她的脸也染成了粉色。你们就这样坐在天台边上，看着太阳一点一点落下去，看着校园的灯一盏一盏亮起来。

"那个纸条……"她突然说，"你要收好哦。"

"会的。"你说，"我会一直带着。"

她靠在你肩上，没有说话。但你能感觉到她的笑容。

*——然后画面开始崩塌——*

不。不只是画面。声音也在消失。色彩也在褪去。

你想要抓住什么——那张纸条、那个约定、那个瞬间——但它们都在从指缝间流走。

只剩下最后一句话，在你脑海中回响：

"不管发生什么，每年今天都要来这里。"

*——记忆中断——*

---

那张纸条……你放进了哪里？你说过会一直带着的。

**记忆深处，有什么东西在隐隐作痛。**
