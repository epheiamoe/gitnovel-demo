# 线索笔记

## 记忆碎片

**天台上的约定**
- 地点：图书馆天台
- 时间：某个黄昏，夕阳西下
- 约定内容：每年今天都要来这里
- 信物：写着"永远在一起"的纸条

**关键对话**
- "不管发生什么，每年今天都要来这里。"——小夏
- "每年今天，我都会在这里等你。"——你

**动作**
- 拉钩：小指勾小指
- 盖章：在额头上轻轻一碰

## 疑问

- 那张纸条现在在哪里？
- 你有没有遵守约定，每年都去天台？
- "不管发生什么"——暗示着有什么事会发生？

## 情感记录

这个约定对你和小夏都很重要。你当时是认真的，你也相信她会认真。

只是……你把它弄丢了。你甚至忘记了它的存在。

## 提示

纸条上写着"永远在一起"。也许它现在还在某个地方……
