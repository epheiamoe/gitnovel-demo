# 故事：秘密的笔记本

*——记忆闪回——*

"我想和你一起写。"

这句话在房间里回荡，比任何声音都清晰。

小夏站在你背后，双手撑在书桌边缘，下巴几乎抵在你的肩膀上。她说话时的气息轻轻拂过你的耳廓，带着一点洗发水的清香。

你的心跳漏了一拍。

"好啊。"你说，声音有点哑，"我们一起。"

然后呢？

你拼命想要回想之后发生了什么。你们有没有找来另一本笔记本？你们有没有真的开始一起写日记？

但什么都想不起来。

记忆在这里断了。像是被撕掉的书页，留下参差不齐的边缘。

你只记得那个瞬间——她说想和你一起写，你的回答是"好啊"。还有她眼睛里的光，像是把整个星空都装了进去。

*——然后是漫长的空白——*

笔记本。深蓝色封面。那本日记。

它去哪了？你们一起写的那些内容，现在在哪里？

*"以后……我想和你一起写。"*

她的声音还在耳边。但你知道，有些"以后"没有来。

---

**你感到一阵强烈的失落感。那本笔记本里写的是什么？为什么你一点印象都没有？**
