# 故事：教室走廊

你站在那张照片前，久久无法移开视线。

那是一张会让你感到陌生又熟悉的脸。熟悉，是因为每天你都会在镜子里看到它；陌生，是因为照片里的那个人的眼神，和你此刻的心境如出一辙。

那种空洞的、不笑的表情。

照片里的你站在樱花树下，阳光从头顶洒下来，樱花瓣落在你的肩上、头发上。明明是那么美的场景，你的眼神却像是穿透了镜头，在看着某个很远很远的地方。

旁边的女生——小夏——正拉着你的袖子说着什么。她的笑容那么灿烂，那么温暖，像是一小团火焰，试图点燃你眼里的光。

"不管发生什么，我都会陪着你。"

那句话像是从照片里浮出来，刻进你的眼睛里。

你忽然想起104教室里的那封信。信里说"永远在你身边"，说"你已经不是一个人"——现在你知道了，那不是空洞的承诺。那是照片里那个笑得像太阳一样的女生，在你最需要的时候，说给你听的誓言。

可是——

为什么照片里的你没有笑？

为什么你的眼神那么空洞？

毕业照应该是大学四年最美好的纪念，是青春的定格，是友谊和爱情的见证。但照片里的你看起来……像是在承受着什么。

你伸手触碰那张照片，指尖触到冰凉的相框。

就在这一瞬间——

一个画面闪过你的脑海。

白色的墙壁。消毒水的气味。某个声音在哭着说什么。

然后是黑暗。

你猛地收回手，心跳快得像是要从胸口跳出来。那是什么？那个画面——白色的墙壁，消毒水的气味——那是医院？那里发生了什么？为什么你会想到那个地方？

头痛欲裂。

你捂住额头，靠在墙上，大口大口地喘着气。周围的一切都变得模糊不清，只有那张照片还在你的视线里定格——照片里的你，和此刻的你，隔着时空遥遥相望。

你到底遗忘了什么？

---

*那个一闪而过的画面——白色的墙壁，消毒水的气味——像是一个危险的信号。*

*那里埋藏着什么？*

*毕业典礼还在等着你。小夏也在礼堂等你。*

*但这张照片里的谜团，似乎比毕业典礼本身更重要。*
