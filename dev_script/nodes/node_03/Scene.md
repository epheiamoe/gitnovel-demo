# 场景：教室走廊

## 地点
教学楼走廊

## 环境
你从104教室出来，沿着走廊继续往前走。

走廊两侧的墙壁上挂满了历届毕业生的合影照片，整整齐齐地排列着，从最老的黑白照片到近些年的彩色照片，像是时光的走廊，记录着这所学校一届又一届的青春。

你的脚步不由自主地慢了下来。

那些照片里的人都笑得很灿烂，穿着那个年代特有的服装，摆着那个年代流行的姿势。每一张照片下面都有一行小字，写着年份和"毕业留影"的字样。

你的目光扫过一张张照片，像是在寻找什么。

然后——

你停住了。

在走廊的中段，有一张照片和其他的略微不同。不是那种整齐划一的合影，而是一张更随意、更生活化的照片。照片的边框有些旧了，颜色也不如其他的鲜艳，像是被人特意取下来过，又重新放回去的。

照片下面写着一行字：

> **2024届毕业留影**

这是去年的照片。

照片里有两个女生站在一起，背景是这所学校的标志性建筑——那棵据说已经有上百年历史的老樱花树。阳光正好，樱花正开，花瓣在风中轻轻飘落。

左边的女生笑得很灿烂，眼睛弯成月牙形，长发在风中微微飘动。她的手拉着右边那个女生的袖子，像是在说什么开心的事情。

右边那个女生……

你盯着那张脸看了很久。

那是你吗？

五官轮廓、眉眼间距、甚至鼻梁的弧度——都和镜子里的自己一模一样。但和你现在的表情不同，照片里的你没有笑。你的眼神有些空洞，像是在看着某个很远的地方，又像是在努力压抑着什么。

明明是毕业照，明明是应该开心的时刻。

你却在微笑。

旁边的女生却笑得很灿烂，像是在努力逗你开心。

照片下方的角落里，用极小的字写着一行字：

> **"不管发生什么，我都会陪着你。——小夏"**
