# 线索笔记

## 记忆碎片

**暴雨中的场景**
- 地点：校门口
- 天气：暴雨
- 状态：你浑身湿透，似乎在雨中等待了很久
- 小夏跑来接你，她的肩膀暴露在雨中

**关键对话**
- "为什么不进去？"——小夏
- "我在等你……"——你
- "我知道……我一直在这里等你……"——小夏

## 疑点

- 为什么你要在暴雨中等待？
- 你在等什么？还是在等谁？
- 为什么小夏说"一直"？

这段记忆的情感浓度很高。你能感受到自己当时的疲惫、固执，还有……某种坚持。

小夏的眼泪让你心疼。她是冒着雨来找你的。
