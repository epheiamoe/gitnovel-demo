# 故事：BE: 独自一人

又是一年春天。

窗外的雨淅淅沥沥地下着，打在窗户上，发出沙沙的声音。你站在窗前，看着玻璃上蜿蜒而下的水痕，思绪不知道飘到了哪里。

公寓的暖气有些老了，嗡嗡地响着，却不怎么暖。你穿着一件单薄的睡衣，站在那里，像是与这个世界隔绝了一样。

手机屏幕亮着。

你刚才又看了一遍那张照片。那个女生又笑了。你把屏幕按灭，但它在你脑海里挥之不去。

她是谁？

你不知道为什么，胸口突然涌起一阵酸涩。那种感觉来得毫无预兆，像是被什么东西猛地击中了一样。你眨了眨眼睛，发现视线有些模糊。

然后你意识到——

你在流泪。

你伸手摸了摸脸，指尖触到了湿润的痕迹。你不知道自己为什么在哭。你不知道那股酸涩是从哪里来的。你只是觉得……失去了什么很重要的东西。

很重要的。

可你连那是什么都不知道。

雨还在下。远处隐约传来汽车的鸣笛声，有人在楼下跑过，踩水的声音很响。你听到隔壁有人在放音乐，隐约能听到歌词——

"如果我们都忘了爱一个人，会怎样……"

你把额头抵在冰凉的玻璃上。

窗外的世界灰蒙蒙的，像是隔着一层纱。你看着自己的倒影，苍白的脸，空洞的眼神，还有……脸颊上不断滑落的泪。

你想起了那张纸条上的字。

"去礼堂前，先去104教室看看。"

104教室。

你从来没有去过。

现在想起来，那里应该有什么东西等着你。但毕业典礼那天，你没有去。你直接去了礼堂。然后典礼结束了。然后……

然后你就醒在了医院里。

那张纸条呢？

你翻遍了房间，找不到了。

窗台上的绿萝已经枯萎了。你不记得上次浇水是什么时候。也许是上周。也许是上个月。时间在你这里变得模糊，失去了意义。

你不知道自己是谁。
你不知道自己失去了什么。
你只知道，很久很久以前，有一个很重要很重要的人，从你的生命里消失了。

而你甚至不记得她的名字。

---

*雨还在下。*
*你靠在窗边，眼泪流干了，却没有动。*
*你就这样站着，站了很久很久。*
