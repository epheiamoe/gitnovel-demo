# 场景：BE: 醒来

## 地点
医院病房，白色的天花板

## 环境

白色。

这是你睁开眼睛后，看到的第一种颜色。

白色的天花板，白色的墙壁，白色的床单，白色的窗帘。空气中弥漫着消毒水的气味，冷冰冰的，刺入鼻腔。窗外有雨声，淅淅沥沥的，像是有人在用指尖轻轻敲打玻璃。

你躺在一张病床上。身上穿着浅蓝色的病号服，手指上夹着某种监测仪器的夹子，冰冷的金属触感让你的皮肤微微发颤。

你转过头。

床边有一把空椅子。

小夏不在了。

只有冰冷的仪器，沉默地闪烁着绿色的光点。还有窗外的雨声，一下一下，像是某种倒计时。

病房的门半开着，走廊里偶尔传来脚步声和轮椅滑过地板的声音。某个地方有护士在说话，声音模糊得听不清内容。

你试图坐起来，却发现全身都没有力气。像是做了一个很长很长的梦，醒来之后却什么都不记得了。

不——

有什么东西在脑海深处挣扎着想要浮出水面。一个名字，一个笑容，一个声音……但当你试图去抓住它的时候，它就像握紧的沙子一样从指缝间流走。

小夏。

你想起了这个名字。但随之而来的是一片空白。

门被推开了。

一个穿着白大褂的医生走进来，表情复杂。他手里拿着一个文件夹，看着你的眼神里有同情，有释然，还有一些你读不懂的东西。

"你醒了。"他说，声音很轻。

他走到床边，翻开文件夹。

"梦境实验已经结束了。"他说，"受试者已成功遗忘所有创伤记忆。"

他把文件递给你。

你低头看向那张纸。

上面写着你的名字，下面是一串复杂的医学术语，还有一行被红笔圈出来的字——

**"创伤记忆已成功封印。建议长期观察。"**

---

## 可见物品
- 手中的文件（上面写着"创伤记忆已成功封印"）
- 床边的空椅子（小夏曾经坐过的地方）
- 窗外的雨
- 闪烁的监测仪器

## 可探索方向
- 病房门外的走廊
- 窗边（可以看雨）
- 枕头下（似乎有什么东西）
