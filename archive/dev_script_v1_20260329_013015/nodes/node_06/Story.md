# 故事：图书馆角落

你站在那张桌前，久久无法移开目光。

2019.09.01。

大一第一天。

那是一切的起点——你的大学生活，你和小夏的故事，还有那些你拼命想要找回的记忆。

你坐下来。

椅子发出轻微的吱呀声，像是在欢迎一个久违的老朋友。阳光从窗户洒进来，照在你的脸上，温暖而柔和。你闭上眼睛，试图让自己沉入记忆的漩涡中。

然后，画面来了。

不是完整的画面，只是一些碎片——

**碎片一：**

阳光照在课桌上。你坐在这里，低着头看一本书——是一本很厚的专业书，你皱着眉头，像是看不懂。桌上刻着字，你的手里拿着一把圆规，犹豫着要不要刻下去。

"在写什么呢？"

一个声音从对面传来。你抬起头——

一张笑脸出现在你的视线里。阳光照在她的侧脸上，她的眼睛弯成月牙形，好奇地看着你。

"我叫小夏。"她说，"可以坐这里吗？"

**碎片二：**

还是这张桌子。窗外的樱花开了又落，落了又开。你们每周三都坐在这里，复习功课，或者只是安静地各看各的书。有时候她会带零食来，分给你一半；有时候她会睡着，醒来时脸上印着书的压痕。

你从未告诉她，每次她睡着的时候，你都会偷偷看她的睡颜。

那是每周三最幸福的时刻。

**碎片三：**

最后一个周三。

你们坐在这里，窗外是即将凋零的樱花。她没有像往常一样看书，只是静静地看着你。

"我们……"她的声音很轻，"我们是不是应该谈谈？"

你想追问她要谈什么。但画面在这里断裂了，像是被人用剪刀剪掉的胶片。

你睁开眼睛。

阳光依旧温暖，窗户上的便签依旧在，桌上的刻痕依旧清晰。但那个"要谈谈"的周三之后发生了什么——你已经不记得了。

也许，那些记忆就是从那时开始被封印的。

---

*你坐在那张椅子上，阳光照在你的脸上。*

*对面那张空着的椅子，似乎还残留着某个人的温度。*

*"我们是不是应该谈谈？"——这句话像是一把钥匙，指向某个你不记得的秘密。*
