# 场景：樱花树下

四月的阳光透过樱花枝叶洒落，在湖面上投下斑驳的光影。

你站在校园的湖边，一阵风吹过，粉色的花瓣纷纷扬扬地飘落，像是下了一场温柔的花雨。湖水平静如镜，倒映着岸边的樱花树和远处的教学楼。

你的目光被吸引到不远处的一棵老樱花树下。那棵树比其他樱花树都要高大，花开得格外灿烂。树下有一张木质长椅，长椅旁边立着一块小小的石碑。

石碑上刻着几行字：

> "愿这片樱花，年年为你盛开。"
> 
> "——小夏 2022.3.28"

落款的日期让你心头一颤——那是你刚上大一的春天。

脑海中突然闪过一些模糊的画面：好像每年这个时候，都有人在这里等你。等待的身影，温柔的笑容，还有……一把撑在头顶的伞。

湖面泛起涟漪，又一阵花瓣飘落，落在你的肩头。
