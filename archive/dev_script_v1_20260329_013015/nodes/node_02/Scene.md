# 场景：神秘的信

## 地点
教学楼104教室

## 环境
你沿着走廊一直走，找到了104教室。

教室的门虚掩着，轻轻一推就开了。没有开灯，整个教室笼罩在一种昏暗的氛围中。只有窗户透进来的阳光，在布满灰尘的空气中切割出一道道光柱。

教室里的桌椅摆放得很整齐，是那种最普通的教室配置——前面是讲台，后面是黑板，一排排课桌椅整齐地排列着。墙上挂着课程表和时间表，纸张已经泛黄，边角卷曲。

但这里显然很久没有人来过了。

你走进去，脚步声在空旷的教室里回响。课桌上积着一层薄薄的灰，仿佛被遗忘在时间的角落里。窗台上有一盆枯萎的绿植，蜷缩成一团，已经彻底失去了生机。

教室正中间的位置，有一张课桌和其他的略微不同。

那是一张靠窗的桌子，位置很好，阳光刚好能照到。桌上放着一个白色的信封，在昏暗的环境中格外显眼，像是黑暗中唯一的灯火。

你走近，拿起那个信封。

信封上写着四个字：

> **给毕业的自己**

笔迹娟秀而熟悉，你不知道为什么，但看到这四个字的时候，心里涌起一种奇怪的感觉——像是某种失去已久的温度，正从记忆的深处缓缓苏醒。

信封没有封口。

你轻轻抽出里面的信纸，展开。
