# 场景：结束

## 地点
无论哪个结局，都在这里汇合

## 环境

"谢谢你玩完了这个故事。"

屏幕上出现了这样一行字。白色的字体，黑色的背景，简洁得像是某种告别。

你盯着那行字看了很久。

然后你的目光落在屏幕的最下方。

那里有另一个名字。

是你的名字。

你盯着那个名字看了很久。你不确定自己应该感到惊讶、困惑，还是释然。也许都有。也许什么都没有。

窗外有鸟鸣声。阳光从窗户照进来，在键盘上投下淡淡的光斑。这是一个普通的下午，和之前的无数个下午一样。

但此刻，一切都不一样了。

"游戏结束了吗？"

有人在问。也许是你。也许是别人。

没有人回答。

屏幕上的光标闪烁了一下，然后——

一行新的字出现了。

"还是……刚刚开始？"

---

## 可见物品
- 电脑屏幕上的文字
- 窗外的阳光
- 键盘上的光斑
- 你的倒影（映在屏幕上）
