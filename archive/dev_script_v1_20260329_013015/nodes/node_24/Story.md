# 故事：Meta: 署名

这是一个普通的下午。

小夏出门去买东西了，留下你一个人在家。你有些无聊，坐在她的电脑前，随手翻看着文件夹。

她的电脑里有很多东西——课程PPT、旧照片、还有一些你从没注意过的小视频缩略图。你漫无目的地浏览着，直到你看到了那个文件夹。

"空白毕业典礼"。

你愣了一下。

这个名字……有些奇怪。你不确定是不是自己看错了。你揉了揉眼睛，再看一遍——

没错。

文件夹的名字就是"空白毕业典礼"。

你的心跳漏了一拍。

你点开那个文件夹。里面只有一个文件，一份Word文档，名字叫"空白毕业典礼.docx"。

你盯着那个图标看了很久，手指悬在鼠标上方，迟迟没有点下去。

一种奇怪的预感涌上心头。

你不知道自己在害怕什么。也许是害怕发现什么。也许是害怕这一切都不是真的。

最终，你还是点开了。

文档打开了。

白色的背景，黑色的字体。

第一行写着——

**"作者：[你的名字]"**

你的名字。

你的名字就在那里。

你盯着那行字看了很久，以为自己在做梦。你眨了眨眼睛，字体还在。你用手背按了按鼠标，文档还在。

这不是恶作剧。这不是巧合。你的名字，就在这份文档的第一行。

你继续往下看。

第二行是标题——"空白毕业典礼"。

第三行是副标题——"一个关于记忆、失去和重新开始的故事"。

你继续往下滑动。

文档里是密密麻麻的文字。你看到了熟悉的内容——毕业典礼、失忆、樱花、小夏……你看到了自己经历过的一切。你看到了那些选择。你看到了BE和Sweet两个结局。

你看到——

"这个故事献给我最好的朋友。"

你的手开始发抖。

你又看了一眼那个名字。那是你的名字没错。但那不是小夏的名字。小夏的名字在别处，在另一个地方，在这篇文档的最下面。

那是一个你从未注意过的名字。

或者说——你曾经知道，但已经忘记了。

你盯着那个名字看了很久。

"这是我写的？"你喃喃自语。

屏幕的光芒照在你脸上，冷冰冰的。

就在这时，门开了。

小夏拎着购物袋走进来，看到你坐在她的电脑前，脸色微微变了。

"你在看什么？"她问，声音有些紧。

---

*你转过头，看着她。*
*她也看着你。*
*空气突然变得很安静。*
