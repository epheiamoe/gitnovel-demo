# 场景：动摇的心

## 地点
礼堂侧门外的小花园

## 环境
"……我们出去走走吧。"

小夏轻声说。她没有等你回答，转身朝礼堂侧门走去。你愣了一下，跟了上去。

侧门外是一个小花园。几棵樱花树零星地分布在草丛中，花期已过，只剩下满地的落红和尚未凋零的花瓣。树下有一张长椅，木质的座位被雨水和岁月侵蚀出斑驳的痕迹。

小夏在长椅的一端坐下，拍了拍身边的位置。

"坐吧。"

你在她身边坐下。两个人之间隔着一个拳头的距离，不远，却也没有更近。

风吹过，樱花树发出沙沙的声响。远处的礼堂里传来阵阵掌声和欢呼声——毕业典礼大概正在进行。但这些都像是另一个世界的事情，和这个安静的小花园无关。

小夏从口袋里掏出一瓶矿泉水，拧开盖子，递给你。

"喝点水吧。你今天早上什么都没吃吧？"

她的语气很平淡，像是在陈述一个事实。你确实不记得自己有没有吃过早餐，胃里空空的，但奇怪的是你并不觉得饿。

小夏双手捧着自己的那瓶水，低着头，看着地面。她的侧脸在阳光下显得很柔和，但你注意到她的睫毛在微微颤抖，像是在忍耐着什么。

沉默持续了很久。

最后，她开口了。

"你真的……什么都不记得了吗？"

她的声音很轻，轻得几乎要被风吹散。她没有看你，只是盯着地面，等待你的回答。

- 地上散落的樱花花瓣
