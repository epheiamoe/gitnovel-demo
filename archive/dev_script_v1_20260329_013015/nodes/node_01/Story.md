# 故事：空白早晨

你不知道自己睡了多久。

再次睁开眼睛的时候，周围的世界已经变得不一样了。那些熟悉的面孔、那些应该记得的名字、那些本该珍藏的回忆——全都消失得无影无踪，仿佛被人用橡皮擦从脑海中彻底抹去。

只剩下无尽的空白。

你坐在那张硌人的塑料椅上，试图回想起任何事情。一件也好。哪怕是自己的名字也好。但什么都没有。只有那片苍白的、无边无际的空白，还有从脑海深处传来的隐隐作痛。

手里的纸条被你攥得更紧了。"小夏"——你默念着这个名字，舌尖仿佛能感受到它的重量。似乎是一个很亲近的人才会用的语气。"有重要的东西"——是什么样的东西，值得在毕业典礼这么重要的日子里，特意留下这样一张纸条？

你想不起来。

周围的嘈杂声变得模糊起来。你听到有人在笑，有人在打电话，有人在叫某个名字……但这些声音都像是隔着一层水幕，遥远得不真实。

只有那张纸条是真实的。

那盏快要坏掉的日光灯又闪了一下，发出令人牙酸的滋滋声。在那一瞬间的明暗交替中，你恍惚看到了一些画面——

图书馆靠窗的位置。

某个人的笑容。

一个声音，很轻很轻地叫着你的名字。

然后，一切消失。

你猛地回过神来，心跳快得像是要从胸口跳出来。那是什么？那是……记忆吗？为什么什么都想不起来？

你深吸一口气，试图让自己平静下来。

毕业典礼的音乐声从远处传来，庄严而有些煽情。也许你应该直接去礼堂——毕竟这是毕业典礼，你应该是来参加毕业典礼的。但那张纸条上的字却在你心里留下了痕迹。

104教室。

那里有什么在等着你呢？

---

*你缓缓站起身，双腿有些发软。周围的人都没有注意到你，每个人都沉浸在自己的世界里。*

*你该往哪里去？*
