# 场景：社团活动室

## 地点
文学社活动室

## 环境
你离开图书馆，沿着校园的小路走向社团活动室所在的那栋老楼。

这是一栋有些年头的建筑，红砖墙面斑驳，爬山虎爬满了整面墙壁。楼道里光线昏暗，脚步声在水泥地面上回响。墙上挂着一些褪色的海报，都是往届社团活动的宣传画。

你在三楼尽头找到了那扇门。

门牌上写着"文学社"三个字，漆面已经有些剥落。门虚掩着，从门缝里透出一丝微弱的光。

你推门进去。

活动室不大，却塞满了东西。四面墙壁都是书架，上面摆满了各种书籍和杂志——有古典文学，有现当代小说，有诗集，还有几摞看起来很有年代感的期刊合订本。窗边放着一张旧沙发，茶几上是几个已经干枯的盆栽。

墙上的相框里装满了照片，都是社团活动的留影。有读书分享会的，有写作比赛的，有户外采风的……每一张照片里都有一些人，笑得很开心。

你的目光落在其中一张照片上。

然后你停住了。

那是一张两个人的合影。背景是这间活动室，墙上贴着"新年快乐"的彩带，桌上摆着一些零食和饮料。照片里的两个人站得很近——

是你和小夏。

她正在为你整理衣领，手指搭在你的领口，动作很轻柔。而你低着头看着她，嘴角带着一丝若有若无的笑意，眼神温柔得像是能溢出水来。

这张照片……你们看起来太亲密了。

不像是朋友。

像是——

你还没来得及细想，目光落在照片下方的角落里，看到了用极小的字写着的一行日期：

> **2022.12.31 新年晚会**

两年前的跨年夜。

你们一起参加了文学社的新年晚会。然后她为你整理衣领，你低头看着她。

那个眼神。

那个笑容。

那是恋人之间才会有的眼神。
