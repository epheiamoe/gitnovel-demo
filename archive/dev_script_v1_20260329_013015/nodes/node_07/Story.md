# 故事：社团活动室

你站在那张照片前，久久无法移开目光。

照片里的那个眼神——那个笑容——那双手轻轻搭在领口上的动作——每一个细节都在告诉你，这两个人之间的关系绝不是"朋友"那么简单。

那是恋人。

是你和小夏。

你的手指不由自主地抬起来，轻轻触碰那张照片。指尖触到冰凉的相框，你却仿佛能感受到那一刻的温度——

画面再次涌来。

**跨年夜。**

活动室里挂着彩带，桌上摆满了零食和饮料。窗外有烟花在绽放，照亮了夜空。你和小夏站在角落里，不知道在说些什么。

她抬起头，看到你的领带歪了。

"等一下。"她说，声音带着一丝嗔怪，"你怎么连领带都打不好。"

她的手抬起来，开始为你整理衣领。动作很轻柔，带着某种理所当然的亲昵。你低头看着她，看着她专注的侧脸，看着她微微颤动的睫毛——

"你知道吗，"她一边整理一边说，声音很轻，"和你在一起的时候，我觉得……特别安心。"

她抬起头，看着你。烟花的余光落在她的眼睛里，像是星星落进了水面。

"谢谢你一直陪着我。"

你的心脏猛地跳了一下。

然后——

画面在这里断裂了。

像是被人用力撕开的纸，边缘参差不齐。有什么东西想要涌出来，却被某种力量死死地按住。

你捂住额头，感受到一阵剧烈的头痛。

但这一次，在疼痛之中，你隐约抓住了什么——

**"和你在一起的时候，我觉得特别安心。"**

**"谢谢你一直陪着我。"**

那不是普通的话语。

那是告白。

小夏在跨年夜对你告白了。而你——

你回应了她吗？

你记不清了。你只记得那一刻她的眼睛那么亮，亮得像是装满了星星。而你的心跳那么快，快得像是随时会冲出胸腔。

然后呢？

然后——

一切归于黑暗。

---

*你站在那张照片前，手指依然悬在相框上方。*

*那个跨年夜，你和小夏之间发生了什么？*

*那之后，你们是恋人了吗？*

*如果是的——*

*那为什么在毕业照里，你的眼神那么空洞？*
