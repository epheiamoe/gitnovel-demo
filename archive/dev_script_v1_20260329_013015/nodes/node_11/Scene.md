# 场景：被遗忘的约定

*——记忆闪回——*

夕阳。

你看到了夕阳。金色的光芒铺满了整个天台，把一切都染成了温暖的橙红色。

你和小夏坐在天台边缘。脚悬在半空，下面是逐渐亮起灯火的校园。风吹过，带来远处操场的喧嚣声和青草的气息。

"我们约定……"

她说着，从口袋里掏出一张小小的纸条和一支笔。她低着头认真地写着什么，夕阳在她脸上投下柔和的光影。

你看着她的侧脸，看着她认真的样子，心里涌起一种奇怪的感觉。好像知道要发生什么，又好像什么都不确定。

"写好了。"

她把纸条递给你。你接过来，打开——

上面写着四个字：

**"永远在一起"**

字迹有些稚嫩，但一笔一划都很用力，好像要刻进纸里一样。

"这是我们的约定。"她抬起头看着你，眼睛里映着夕阳的光，"不管发生什么，每年今天都要来这里。"

你把纸条折好，小心翼翼地放进口袋。

"好。"你说。

她笑了，笑得很开心。然后她伸出手，小指勾住了你的小指。

"拉钩。"

*——记忆闪烁——*

画面开始不稳定。夕阳、天空、她的脸……都在晃动。

但你听到了自己的声音——

"每年今天，我都会在这里等你。"

---

**场景**: 记忆闪回——图书馆天台
**时间**: 某个黄昏
**状态**: 记忆不稳定，但核心约定依稀可辨
