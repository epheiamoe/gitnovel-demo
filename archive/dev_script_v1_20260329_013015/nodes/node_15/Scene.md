# 场景：真相浮现

礼堂后台的杂物间里，灰尘在昏暗的灯光下浮动。

你站在一张堆满杂物的桌子前。桌上铺着各种旧文件、破旧的装饰品，还有一盏摇摇欲坠的台灯。

小夏站在你身后，没有说话。只是静静地看着你的背影，像是在等待什么。

你低头看着手中的文件夹。封面上"梦境协议"四个字像是烙印一样，灼烧着你的视线。

你深吸一口气，打开了它。

第一页是一份申请表。抬头写着：

**"梦境重构计划——记忆选择性遗忘服务"**

下面是密密麻麻的条款。你扫视着那些文字，每一个词都像是冰锥一样刺入你的眼睛——

> **"申请者自愿接受记忆重构，以避免创伤性记忆带来的伤害。"**
> 
> **"本服务通过介入REM睡眠周期，实现特定记忆的选择性模糊化。"**
> 
> **"费用由梦境科技有限公司承担。"**

申请日期：2024年6月15日
申请者签名：________________

那个签名。你认得那个笔迹。

是你自己的。

---

"那天晚上……"

小夏的声音从身后传来，很轻，很颤抖。

"你崩溃了。一直哭，一直说不想记起来……"

她的声音哽住了。

"你说，那些记忆太痛了。你说，你不想让我看到你那个样子。你说……"

她深吸一口气。

"你说你宁愿忘记。"
