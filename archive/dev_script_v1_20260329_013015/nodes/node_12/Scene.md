# 场景：秘密的笔记本

*——记忆闪回——*

台灯的光晕在黑暗中铺开，照亮了一小片书桌。

你坐在宿舍的书桌前，面前摊着一本厚厚的笔记本。封面是深蓝色的，边角有些磨损——已经被翻过很多次了。

你在写日记。笔尖在纸上沙沙作响，写下今天的流水账。

"又在写那些秘密？"

一个声音从背后传来，带着几分好奇和撒娇。你还没反应过来，就感觉到一个温热的重量靠在了你背上。

小夏从背后探头过来，想要看笔记本上的内容。

"不给看。"你笑着把本子合上，护在胸口。

"哼，小气鬼。"她撅着嘴，但没有再抢，"我又不会告诉别人。"

你看着她故作委屈的样子，忍不住笑了。"以后……"她的声音突然变得很轻，轻到你几乎听不清。

"以后怎么？"你问。

她没有看你，只是盯着自己的手指，声音像是蚊子哼：

"以后……我想和你一起写。"

房间里很安静。窗外的蝉鸣声忽远忽近。台灯的光把你的影子投在墙上，两个人的轮廓重叠在一起。

"好啊。"你说，"我们一起。"

她抬起头，眼睛亮晶晶的，像是盛满了星星。

*——记忆开始变得模糊——*

但画面没有继续。像是有人按下了暂停键，把这个瞬间定格在了这里。

你试图回想"之后"——你们有没有真的一起写过那本日记？那本笔记本现在在哪里？

但你想不起来。
