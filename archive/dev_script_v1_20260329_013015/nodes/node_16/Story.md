# 故事：毕业典礼

小夏在往前走。

一步，两步，三步……

你看着她的背影。学士服的下摆在膝盖处轻轻晃动，马尾辫随着她的步伐一晃一晃的。

她没有回头。

但你知道她在等你。

礼堂里很吵。有人在笑，有人在哭，有人在大声喊着同学的名字。但这些声音都像是从很远的地方传来，模糊而失真。

你的脑海中闪过很多画面。

樱花树下的石碑。暴雨中的校门口。天台上的约定。那本写满秘密的日记。黑暗中的拥抱。

*"就算忘记了，我也会替你记得。"*

还有那份协议。还有那句"如果记得，你真的会死"。

所有的一切都在这一刻涌上来，撞击着你的心脏。

小夏已经走到舞台边缘了。她还是没有回头。但她的脚步慢了下来。

她在等。

她说过，不管你怎么选，她都在。

但她还是在等。

---

毕业证书还空着。名字还差一个签名。你可以选择继续坐在这里，也可以选择冲上去。

小夏在舞台边缘等待着。

人群在窃窃私语。

你该怎么做？

**这是她给你的承诺。也是她给你的选择权。**
