# 场景：宿舍夜晚

## 地点
宿舍楼走廊，晚上

## 环境
你离开社团活动室，沿着来时的路往回走。

夜幕已经降临。校园里的路灯亮了起来，昏黄的光在柏油路上投下一个又一个圆形的光斑。远处有蝉在鸣叫，此起彼伏，像是一首单调的夏日催眠曲。

你不知不觉走到了宿舍楼。

宿舍楼是一栋六层的旧楼，外墙刷着淡黄色的涂料，在夜色中显得有些昏暗。你站在楼下，抬头看着那些亮着灯的窗户——有的窗户透出暖黄色的光，有的则是冷白色的荧光。

三楼走廊尽头的门上贴着一张纸条。

你走近，撕下那张纸条。上面用熟悉的笔迹写着：

> **318 - 你的房间**

这是你的宿舍。

你推开门。

房间不大，但收拾得很整洁。一张单人床靠在墙边，床上的被褥叠得整整齐齐。书桌靠窗，上面摆着一台旧笔记本电脑，几本摞起来的书，还有——

一本日记本。

那是一本深蓝色封皮的日记本，封面有些磨损，边角卷曲，显然被翻阅过很多次。日记本上着一个小小的铜锁，锁孔里插着一把小钥匙。

你拿起日记本，试图打开——

但锁住了。

你试着拔动那把小钥匙，但它纹丝不动。像是被什么东西粘住了，又像是锁芯已经生锈。

你把日记本放下，目光落在书桌上。

书桌的一角立着一个相框。相框里是一张照片——你和一个小女孩的合影。照片很旧了，边角有些泛黄，像是很多年前的照片。小女孩大概七八岁的样子，扎着两个羊角辫，笑得眼睛都眯成了一条缝。而你——

你也笑得很开心。

照片背面写着一行字：

> **和小夏的第十年**

你盯着那行字，心脏像是被什么东西狠狠击中了。

第十年。

你们认识十年了？

那意味着……在大学之前，你们就已经认识了？

但为什么……你一点印象都没有？
