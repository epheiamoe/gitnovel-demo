# 故事：宿舍夜晚

你坐在书桌前，把那张照片拿起来，凑近台灯的光仔细端详。

照片里的两个人笑得像是一对傻瓜。小女孩的眼睛弯弯的，露出两颗小虎牙；而少年——那是你——的笑容里带着某种宠溺的温柔，像是在看世界上最重要的宝贝。

"和小夏的第十年。"

那是什么意思？

如果现在是2025年大学毕业，那么十年前应该是2015年。那时候的你应该是……十五岁，刚上高一。而小夏呢？也是十五岁？

你们是高中同学？

不，不对。

照片里的小女孩看起来只有七八岁。如果是2015年，她应该才三四岁……

你皱起眉头，试图理清时间线。

现在是2025年，你们认识十年的话，就是2015年。但如果小夏现在22岁左右（大学毕业生），十年前她应该是12岁……

照片里的她看起来不像12岁。

这太奇怪了。

你的目光落在那张照片上，看着那个笑得像个小太阳的小女孩。她是谁？为什么你们的合影会写着"和小夏的第十年"？

除非——

一张新的照片从日记本里滑落出来，飘到地上。

你弯腰捡起来。

那是一张更旧的照片，边角都已经卷曲发黄了。照片上是两个小孩子，一男一女，大概四五岁的样子，站在一棵很大的樱花树下。女孩子靠在男孩子的肩膀上，男孩子的手搭在女孩子的头上。

两个小孩子的脸上都脏兮兮的，像是刚玩完泥巴回来。

照片背面写着一行字，字迹歪歪扭扭的，像是小孩子写的：

> **2022.03.28 和小夏的第八年**
> **樱花树下的约定——要做一辈子好朋友！**

2022年3月28日——那是大三下学期。

但照片里的小孩子明显是四五岁的模样……

除非你们真的是从小就认识。

除非你们在很小很小的时候就认识了——然后在某个时刻，她从你的记忆里消失了——然后又在大学图书馆里，以"陌生人"的身份，重新出现在你的生命里。

你的手开始发抖。

如果这是真的……

那"小夏"这个名字，不只是一个恋人的名字。

她是你从小到大的青梅竹马。

而你——把她忘记了。

---

*你坐在那张照片前，手里还攥着那张"第八年"的照片。*

*"要做一辈子好朋友"——那是小孩子的誓言。*

*但你们做到了吗？*

*你们不只是好朋友……你们变成了恋人。*

*但现在，你连她是谁都不记得了。*

*而那本锁着的日记本里……也许藏着所有的答案。*
