# 场景：碎片拼凑

毕业典礼的礼堂后台，昏暗的灯光在走廊尽头摇曳。

你不知道自己是怎么走到这里的。也许是跟着小夏，也许是被某种直觉牵引。周围堆满了落灰的桌椅和被丢弃的装饰道具，空气中有一股陈旧的气息。

小夏站在你面前，低着头，双手紧紧握在一起。她的肩膀在微微发抖。

你们沉默了很久。

终于，你开口了。

"我想起来了……"你说，声音有些沙哑，"一些片段。"

她猛地抬起头，眼眶红了。泪光在昏暗的灯光下闪烁，像是碎掉的星星。

"樱花树下的石碑。"你继续说，"暴雨中在校门口等我的人。还有……天台上写下的约定。"

每一个词，都像是从记忆的碎片中打捞上来的贝壳。破碎，却依然闪着光。

"每年今天都要去那里。"你说，"'永远在一起'。还有那本日记……"

你的声音开始颤抖。

"我答应过要和你一起写的。"

小夏的眼泪终于落了下来。她没有说话，只是用力握紧了你的手。

她的手心是凉的，指尖在发抖。但那个握紧的力度，比任何语言都更用力。

"我想起来了。"你说，"你抱着我说，就算我忘记了，你也会替我记得。"

你看着她的眼睛。

"我记得你。"
