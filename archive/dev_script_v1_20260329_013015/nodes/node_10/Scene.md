# 场景：暴雨之日

*——记忆闪回——*

画面模糊而摇晃，像是被水浸透的老照片。

你看到了校门口。雨下得很大，豆大的雨点砸在地面上，溅起一朵朵水花。天地间灰蒙蒙的一片，只有远处的路灯在雨幕中晕开微弱的光。

你站在校门口的屋檐下，浑身湿透。不知道为什么，你没有躲进去，而是站在雨里，像是在等什么。

脚步声。

有人跑过来，脚步声在雨水中啪嗒啪嗒作响。一把伞举到你头顶，挡住了倾盆而下的雨水。

"为什么不进去？"

是女孩的声音，带着哭腔。

你抬起头，透过模糊的视线看到一张熟悉的脸。小夏站在你面前，她的半边身子已经湿透，制服裙摆贴在腿上，但她浑然不觉，只是焦急地看着你。

"我在等你……"

你听到自己的声音，沙哑而疲惫。

"你全身都湿了！"她的声音颤抖着，"你傻不傻？下这么大的雨，你站在这里干什么……"

"我在等你。"

你重复了一遍。声音很轻，但很坚定。

她的眼眶红了，泪水和雨水混在一起，从脸颊滑落。

"我知道……"她说，"我一直在这里等你……"

*——记忆断裂——*

画面猛地碎裂，像是被什么东西强行切断。
