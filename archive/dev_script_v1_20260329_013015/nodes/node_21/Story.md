# 故事：BE: 空白之后

日子一天天过去。

你一个人住在这间小公寓里。房租很贵，但你付得起——父母留下的钱足够你挥霍一阵子。窗台上有一盆绿萝，是搬进来的时候买的，叶子绿油油的，长得很好。

你每天做着差不多的事。

起床。洗漱。出门。上班。处理那些枯燥的文件和邮件。中午吃便利店的便当。晚上下班，在楼下的快餐店随便吃点。然后回家，躺在沙发上，对着天花板发呆。

然后洗澡，睡觉。

周而复始，像是某种精密的机器在运转。

你不怎么说话。在公司里，你需要说话的时候会正常说话，但除此之外，你不会多说一个字。同事觉得你性格内向，但也仅此而已。没有人知道你失去了什么。

因为你自己也不知道。

你偶尔会想起一些碎片。

樱花。很美的樱花，粉色的，落在肩上的，像雪一样。
雨。很大很大的雨，打在窗户上，看不清外面的世界。
图书馆的窗户。阳光从那里照进来，空气中有一股旧书的味道。
还有一个笑容。很温柔很温柔的笑容，像是春天的风。

但你抓不住它们。

每当你想仔细去看那些画面，它们就像握紧的沙子一样从指缝间流走。你只能眼睁睁地看着它们消失，剩下手指间残留的一点点触感。

然后，连触感也消失了。

今天整理手机的时候，你翻到了一张照片。

照片里有两个人。

一个是你——笑容有些僵硬，像是不知道该怎么做表情。旁边是一行小字：毕业典礼，2024年6月15日。

另一个是一个女生。长头发，眼睛弯弯的，笑得很开心。她挽着你的手臂，头靠在你肩上。那个姿势很自然，很亲密，像是做过无数次一样。

你盯着那张照片看了很久。

照片的日期是两年前。

但你想不起她的名字。

她是谁？

她为什么挽着你的手？

她为什么靠在你肩上笑得那么开心？

你试图在手机里搜索她的名字，但没有找到。微信里有几百个好友，但你一个一个翻过去，都对不上那张脸。通讯录里也没有特别的备注。

她就那样消失在你的记忆里，像是从未存在过一样。

只剩下这张照片。

证明她曾经存在过。证明你们曾经那么亲密过。

---

*你把照片放大，又缩小。放大，又缩小。*
*最后，你盯着那个笑容看了很久很久。*
*然后你关掉了手机。*
