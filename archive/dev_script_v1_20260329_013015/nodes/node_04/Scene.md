# 场景：小夏的出现

## 地点
礼堂入口

## 环境
你终于走到了礼堂入口。

礼堂的大门敞开着，里面传来阵阵人声和音乐声。红色的帷幕高高挂起，舞台上布置着鲜花和气球，横幅上写着"2025届毕业典礼"的字样。阳光从高高的窗户洒下来，照在穿着学士服的人群身上，给每个人都镀上了一层金色的光芒。

毕业的气息扑面而来。

但你刚想迈步进去——

"等一下。"

一个声音从身后传来。

你转过身。

一个女生站在走廊的光影交界处，逆着光，看不清表情。但你能看到她的轮廓——纤细的身影，长发披肩，穿着一件有些皱的学士服。

她朝你走来，脚步越来越快，最后几乎是跑过来的。

然后她停在你面前，抬起头看着你。

你看清了她的脸。

那是一张会让你心跳漏拍的脸。不是因为她有多美——虽然她确实很美——而是因为看到她的那一刻，你的心脏像是被什么东西狠狠攥了一下。

她的眼眶有点红。

"你终于来了……"

她的声音有些沙哑，像是哭过，又像是在努力忍住不哭。

"我找了你很久。"

她站在那里，看着你，眼睛里有泪光在闪烁。她张了张嘴，似乎想说什么，却又不知道从何说起。

最后，她只是轻轻地问了一句：

"你……还好吗？"

她的目光落在你身上，小心翼翼的，像是在看一件易碎的珍宝。她的手抬起来，似乎想要触碰你，却在半空中停住了，像是怕你会碎掉一样。

阳光从她身后洒进来，照亮了她眼角的泪痕。

而你只能站在那里，一脸茫然地看着她。

你不认识她。

但你的心脏在狂跳。

---

## 可见人物
- 小夏：穿着皱巴巴的学士服，长发披肩，眼眶微红

## 可探索方向
- "你是谁？"
- "我们……认识吗？"
- （沉默地看着她）
