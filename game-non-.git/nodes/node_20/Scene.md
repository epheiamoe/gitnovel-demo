# 场景：交汇

## 地点
无论哪个结局，你们都站在这里

## 环境

毕业典礼结束了。

你站在礼堂外的分岔路口——不，也许不是礼堂外。也许是樱花树下。也许是某个你从未见过的地方。

阳光照在你们身上。

你低头看向自己的手。小夏的手就在那里，温暖而真实。不管发生过什么，不管未来会发生什么，此刻，她在你身边。

这四年来的一切——甜蜜的、痛苦的、真实的、虚幻的——都沉淀在这里。它们像是河底的鹅卵石，被时间的流水冲刷得光滑圆润，却始终在那里。

你曾经拥有过什么？你曾经失去过什么？

也许你已经记不清了。也许你从来都不需要记清。

因为此刻比记忆更重要。

因为她就在这里。

小夏的手在你手心里，微微动了动。她抬起头看着你，眼神里有询问，也有一丝不安。

"怎么了？"她轻声问。

风吹过，带起几片花瓣。它们落在你们肩上，落在你们交握的手上，落在脚下的路上。

路还很长。

但你们会一起走下去。

---

## 可见物品
- 小夏的手
- 落在肩上的花瓣
- 脚下的路（向前延伸，看不到尽头）
- 头顶的阳光

## 可探索方向
- 继续往前走（前方的路）
- 回头看看（过去的路）
