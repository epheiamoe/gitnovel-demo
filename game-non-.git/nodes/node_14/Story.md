# 故事：碎片拼凑

泪水从她的脸上滑落，滴在你们交握的手背上。

小夏没有说话。她只是哭。无声地、用力地、像是把四年的眼泪都在这一刻流尽。

你看着她的样子，心里某个地方隐隐作痛。

"对不起。"你听到自己说。

她摇摇头，哽咽着："不是你的错……从来都不是你的错……"

"是我自己选的。"她说，声音断断续续，"是你自己选的……"

你愣住了。

"什么……"

她终于抬起头，看着你。泪眼朦胧，却有一种如释重负的表情。

"你想起来了一部分。"她说，"但不是全部。"

她松开你的手，后退了一步。从身后的架子上，她拿起了什么东西。

那是一份文件夹。封面上的字让你的心跳漏了一拍——

**"梦境协议"**

"有些事……"她的声音很轻，"我应该告诉你了。"

她的手在发抖，但还是把文件夹递到了你面前。

"但在那之前。"她深吸一口气，"我想让你知道——"

"不管你记起了什么，不管你接下来怎么选……"

她的眼睛直视着你，泪水还挂在睫毛上，但目光前所未有的坚定。

"我都在。"

---

**四年的秘密，即将揭晓。**

**你手中握着那份文件夹，沉甸甸的。**
