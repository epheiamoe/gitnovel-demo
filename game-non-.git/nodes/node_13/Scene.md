# 场景：小夏的眼泪

*——记忆闪回——*

黑暗。

什么都看不见。只有无尽的黑暗，和压抑的哭声。

你在哭。

不，不是"你在哭"。是你"看到"自己在哭。你站在某个地方，看着另一个自己蜷缩在角落里，肩膀剧烈地颤抖。

周围一片漆黑。像是深夜，像是噩梦，像是世界末日。

"我不想忘记……"

你听到自己的声音，沙哑而绝望。

"我不要忘记……"

那是哭腔。是崩溃边缘的声音。是某个人在溺水时发出的最后求救。

然后，有人抱住了你。

是小夏。

她什么都没说。只是紧紧地抱着你，用手轻轻拍着你的背。一下，两下，三下……像是在安抚一只受惊的小动物。

"我不想忘记那些……"你听到自己的声音从她的怀里传出来，闷闷的，破碎的，"我不想忘记你……"

她抱得更紧了。

"没关系。"

她的声音很轻，轻到几乎听不见。但你听到了。

"就算忘记了……"她说，声音在颤抖，"我也会替你记得。"

*——记忆开始崩塌——*

黑暗在吞噬一切。小夏的脸、她的声音、她的温度——都在远去。

你想要抓住她。你不想让她消失。

但记忆像沙子一样从指缝间流走。

最后听到的，是她的哭声。和你的哭声混在一起，纠缠在一起，然后一起消失在黑暗中。
