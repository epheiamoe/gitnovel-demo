# 故事：Sweet: 新的开始

阳光从窗帘的缝隙透进来，在地板上画出一条金色的线。

你睁开眼睛。

天花板上有昨晚台灯的光晕，还残留着一点点温暖的橘色。空气里有咖啡的香味，淡淡的，从客厅的方向飘过来。远处有水声——她在洗碗。

你侧过头。

枕边有几根长发，是她的。枕头上还留着她的气息，淡淡的洗发水味道，混合着某种让你安心的温度。你想起昨晚她说的话——"睡吧，我会在这里"——嘴角不自觉地微微上扬。

你突然想起了一切。

好的，坏的，所有的。

那些甜蜜的清晨，你们一起在食堂吃早餐。她总是把自己吃不完的油条塞给你，你嘴上抱怨却每次都接过来。

那些安静的下午，你们在图书馆并肩看书。阳光从窗户照进来，她的头发被染成金色，你偷偷看她，被她抓个正着。

那些争吵的夜晚，你们红着眼眶却谁都不肯低头。冷战持续了整整三天，最后还是你先认输。

那场暴雨。

你闭上眼睛。那段被封印的记忆慢慢浮现，像是从水底浮上来的气泡——

"你为什么不肯告诉我？！"

她的声音在雨里几乎是嘶吼的。雨水打在她脸上，分不清是雨还是泪。

"你以为你一个人扛着就了不起吗？你以为你把我排除在外就是保护我吗？"

她哭着问你，你却什么都没说。你只是站在那里，像一块石头。

因为你知道。

你知道自己的病，知道那个实验，知道自己可能随时会忘记一切。你不想让她在你忘记之后还要承受失去的痛苦。所以你选择了独自面对，选择了沉默，选择了——

把她推开。

那个记忆太沉重了。你曾经拼命想要逃避它，逃避那些愧疚和自责。所以你选择了封印，选择了遗忘。

但现在，你不再害怕了。

浴室的门开了。

小夏披着湿漉漉的头发走出来，身上穿着你的旧T恤——她总是嫌自己的睡衣不好看，却喜欢穿你的。她看到你醒了，眼睛弯成月牙。

"你醒了？"她轻声说，"我还以为你会多睡一会儿。毕业之后你一直很累，难得休息。"

她走到床边，弯下腰，额头轻轻抵着你的额头。

她的头发还没擦干，水珠滴在你脸上，凉凉的。

"早安，"她说，微微笑了，"今天，我们去哪里？"

你看着她的眼睛。那双眼睛里有笑意，有温柔，还有一点狡黠——她知道你会说什么。

窗外，阳光正好。

*新的故事，从这里开始。*
