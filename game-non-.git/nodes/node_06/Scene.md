# 场景：图书馆角落

## 地点
校园图书馆，靠窗的角落座位

## 环境
你告别小夏，独自一人来到了图书馆。

图书馆是一栋三层的建筑，外表是古老的红砖墙，爬满了常青藤。推开门，一股纸张和木头特有的气息扑面而来。阳光透过高大的窗户洒进来，照亮了漂浮在空气中的尘埃，让整个空间笼罩在一种温暖的、金色的光芒中。

这里很安静。偶尔有翻书的声音，有键盘敲击的声音，有脚步声轻轻走过。但没有人说话。每个人都在自己的世界里忙碌着。

你沿着书架之间的过道往里走，脚步不由自主地放轻了。像是被这里的氛围感染，又像是……某种本能的指引。

然后，你看到了那个角落。

那是一个靠窗的位置。一张单人桌，一把椅子，窗外是一棵老樱花树的枝丫。阳光正好从窗户照进来，在桌面上投下斑驳的光影。

你走过去，像是被什么力量牵引着。

桌面上刻着一些字迹。

你弯下腰，仔细辨认。那是一行用小刀或者圆规刻出来的字，笔迹歪歪扭扭，却带着某种稚拙的认真：

> **2019.09.01 大一第一天**

这是……你第一天来大学时刻的？

你伸手抚过那些字迹，指尖触到木头的纹理。字迹已经很旧了，刻痕的边缘已经磨平，但依然能够辨认。2019年9月1日——那是四年前的今天，也是你大学生活的起点。

你抬起头，看向窗户。

窗户上贴着一张泛黄的便签纸，用透明胶带仔细地粘着，像是怕它会掉落。便签上写着几个字，笔迹很熟悉——是你自己的：

> **记得每周三来这里占座哦**

每周三。占座。

你不知道为什么，但这句话让你的心脏猛地跳了一下。有什么画面在脑海里一闪而过——

每周三的阳光。每周三的风。每周三坐在你对面的那个人。

你的目光落在对面的椅子上。那张椅子是空的，桌面也很干净。但你仿佛能看到一个模糊的身影坐在那里，低头看书，偶尔抬头看你一眼，然后笑。

你认识那个笑。

那是小夏。
