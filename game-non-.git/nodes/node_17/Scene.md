# 场景：最后的选择

## 地点
毕业典礼礼堂外，分岔路口

## 环境

毕业典礼的音乐渐渐远去。

你站在礼堂外的分岔路口，周围的人流慢慢散去。阳光从头顶洒下来，却让你感到一阵不真实的眩晕。手里还攥着那张毕业证书，红色的封皮烫着金字，沉甸甸的。

小夏站在你面前。

她今天化了淡妆，穿着一件浅蓝色的连衣裙，外面套着学士服。阳光照在她的发丝上，泛着柔和的光泽。四年来，你见过她无数次的样子——但此刻的她，美得让你几乎不敢直视。

"典礼结束了。"她开口，声音很轻，像是怕惊扰什么。"接下来，你想怎么做？"

她顿了顿，眼神闪烁了一下。那一瞬间，你看到了期待，也看到了一丝恐惧。

"如果你想知道全部真相……"她的声音更轻了，"我可以告诉你。但这四年发生的一切……"她低下头，看着自己的脚尖。"那可能会很痛苦。你确定吗？"

她重新抬起头，看着你。那双眼睛里，有太多你读不懂的情绪。

"或者——"她深吸一口气，似乎在鼓起勇气，"我们可以重新开始。"

风吹过，带起几片不知从哪里飘来的花瓣。毕业典礼的红色气球还绑在远处的柱子上，在风中轻轻摇晃。

你们就这样站在阳光下，站在过去的终点和未来的起点。

---

## 可见物品
- 手里的毕业证书
- 小夏手中也拿着相同的毕业证书
- 远处飘散的红色气球
- 分岔路口的两条路：一条通往樱花树的方向，一条通往未知

## 可探索方向
- 樱花树的方向（隐约能看到远处的粉色）
- 未知的那条路（小夏似乎在等待你的回答）
