# 故事：最后的选择

典礼结束了。

你站在礼堂外的分岔路口，手里攥着那张毕业证书。红色的封皮在阳光下闪闪发光，上面的金字——你的名字——此刻看起来却那么陌生。

小夏就站在你面前。

她今天很美。你不知道为什么，在失去所有记忆之后，你依然能感受到这一点。也许是因为她看向你的眼神——那种眼神，只有在看着极其重要的人时才会出现。

"典礼结束了。"她轻声说，"接下来，你想怎么做？"

你沉默着，等待她继续说下去。

她深吸一口气。"如果你想知道全部真相……"她的声音有些颤抖，"我可以告诉你。"

"全部的真相？"

"嗯。"她点点头，眼眶有些发红。"关于这四年。关于你为什么失忆。关于……关于我们之间发生的一切。"

她顿了顿，像是在组织语言。

"但是……"她的声音更轻了，"那些记忆可能会很痛苦。你曾经经历过一些……不太好的事情。我帮你封印它们，是因为我不想看到你再——"

她突然停住了，像是意识到自己说得太多。

"……算了。"她摇摇头，挤出一个笑容。"总之，如果你想知道，我可以全部告诉你。但这意味着……那些痛苦也会跟着回来。"

她抬起头，直视着你的眼睛。

"或者，我们可以重新开始。"她说，"忘掉那些。重新开始。就像……就像我们刚认识的时候那样。"

风吹过，带起几片花瓣。阳光照在她的脸上，你看到她的睫毛在微微颤动。

她在等待你的回答。

---

*小夏的手垂在身侧，指尖轻轻蜷缩着，像是在压抑什么。*
