# 故事：BE: 醒来

白色。

这是你睁开眼睛后，看到的第一种颜色。

白色的天花板，白色的墙壁，白色的床单，白色的窗帘。空气中弥漫着消毒水的气味，冷冰冰的，刺入鼻腔。窗外有雨声，淅淅沥沥的，像是有人在用指尖轻轻敲打玻璃。

你躺在一张病床上。

身上穿着浅蓝色的病号服，手指上夹着某种监测仪器的夹子，冰冷的金属触感让你的皮肤微微发颤。

你转过头。

床边有一把空椅子。

小夏不在了。

只有冰冷的仪器，沉默地闪烁着绿色的光点。还有窗外的雨声，一下一下，像是某种倒计时。

你试图坐起来，却发现全身都没有力气。像是做了一个很长很长的梦，醒来之后却什么都不记得了。

不——

有什么东西在脑海深处挣扎着想要浮出水面。一个名字，一个笑容，一个声音……但当你试图去抓住它的时候，它就像握紧的沙子一样从指缝间流走。

小夏。

你想起了这个名字。但随之而来的是一片空白。

门被推开了。

一个穿着白大褂的医生走进来，表情复杂。他手里拿着一个文件夹，看着你的眼神里有同情，有释然，还有一些你读不懂的东西。

"你醒了。"他说，声音很轻，"梦境实验已经结束了。"

他走到床边，翻开文件夹。

"受试者已成功遗忘所有创伤记忆。"他念道，语气平淡得像是在读天气预报。

他把文件递给你。

你低头看向那张纸。

上面写着你的名字，下面是一串复杂的医学术语——什么"海马体"、"前额叶皮层"、"神经递质"……你看不懂，也不想看懂。

但有一行字被红笔圈了出来，每个字都像是刀子刻进眼睛里——

**"创伤记忆已成功封印。"**

**"建议长期观察。"**

你盯着那行字看了很久。

然后你问："那些记忆……是真的吗？"

医生沉默了一会儿。他似乎在斟酌用词。

"从医学角度来说，"他最终说，"是的。它们曾经是真实的经历。但现在，它们已经被成功地……封存起来了。"

"封存。"

"对。"他点点头，"你不会再感受到那些痛苦了。这对你来说是好事。"

他说完这句话，就转身走向门口。

"好好休息。"他在门口停下，头也不回地说，"她会来看你的。"

门关上了。

雨还在下。

---

*你躺在病床上，手里握着那份文件。窗外灰蒙蒙的天光透进来，照在你苍白的脸上。*

*你不知道小夏是谁。但那个空椅子告诉你，她曾经在这里。*

*很久，很久。*
