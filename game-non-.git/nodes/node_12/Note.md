# 线索笔记

## 记忆碎片

**场景**
- 地点：宿舍
- 时间：某个夏夜
- 物品：深蓝色笔记本，你用来写日记

**关键对话**
- "又在写那些秘密？"——小夏
- "以后……我想和你一起写。"——小夏
- "好啊，我们一起。"——你

**小夏的状态**
- 从背后靠过来
- 下巴几乎抵在你肩膀上
- 眼睛亮晶晶的

## 疑点

- 那本笔记本现在在哪里？
- 你们有没有真的一起写？
- 记忆为什么在这里断裂？"之后"发生了什么？

## 情感记录

这个瞬间很甜蜜。你能感受到当时自己心跳加速的感觉。

但之后的事，你什么都不记得了。那段空白太长了。

## 提示

笔记本、约定、秘密……这些关键词似乎都与你的记忆缺失有关。

也许你应该试着回忆更多……或者，也许有些事情你本能地想要遗忘。
