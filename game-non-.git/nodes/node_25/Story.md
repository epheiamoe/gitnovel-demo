# 故事：结束

"谢谢你玩完了这个故事。"

屏幕上出现了这样一行字。

白色的字体，黑色的背景，简洁得像是某种告别。你盯着那行字看了很久，不知道该有什么感觉。

然后你的目光落在屏幕的最下方。

那里有另一个名字。

是你的名字。

你盯着那个名字看了很久。你不确定自己应该感到惊讶、困惑，还是释然。也许都有。也许什么都没有。

这是你写的故事吗？

如果是，那你是在什么时候写的？在大一？大二？或者更早？你不记得了。你只知道，这个故事里有太多你不知道的东西——那些细节，那些选择，那些BE和Sweet的分支……

它们是你创造的吗？

还是你只是……经历了它们？

你想起小夏。她知道吗？她看过这份文档吗？她知道自己的"故事"被这样写下来吗？

或者——

或者小夏也是作者之一？

你不知道。你只知道，你很想知道答案。

窗外有鸟鸣声。阳光从窗户照进来，在键盘上投下淡淡的光斑。空气里有咖啡的味道，混合着某种让你安心的气息。

这是一个普通的下午，和之前的无数个下午一样。

但此刻，一切都不一样了。

"游戏结束了吗？"

有人在问。也许是你。也许是别人。

没有人回答。

屏幕上的光标闪烁了一下，然后——

一行新的字出现了。

"还是……刚刚开始？"

那行字在屏幕上停留了几秒钟，然后慢慢消失。

取而代之的是两个选项。

[重新开始]

[结束游戏]

光标在它们之间跳动，等待着你的选择。

---

*你看着屏幕。*
*阳光照在你背上，暖暖的。*
*你知道，无论你选择什么——*
*故事，已经开始了。*
