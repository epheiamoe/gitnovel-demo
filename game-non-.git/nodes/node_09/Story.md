# 故事：樱花树下

你走近那块石碑，指尖轻轻触碰刻字的凹痕。石碑的边缘已经被岁月打磨得光滑，但字迹依然清晰。

"小夏 2022.3.28"

那是大一春天的日期。你们认识……四年了？从一开始就是她。

一阵风吹过，樱花瓣落在你的发间。恍惚间，你好像看到了什么——

*一个女孩站在这棵树下，踮起脚尖往远处张望。*
*她手里拿着一封包装精致的信，指尖有些发白。*
*看到你的身影出现在湖边小路的尽头时，她的眼睛亮了起来。*

画面一闪而逝。

你猛地回过神，心跳漏了一拍。那一刻的情感太过强烈，强烈到你几乎要想起什么——但那层薄薄的迷雾依然笼罩着你的记忆。

湖边的长椅上空无一人。远处隐约可以看到礼堂的屋顶，那里正在进行毕业典礼的准备工作。

樱花还在飘落。

每年今天，都有人在这里等你。

*而你……好像从来没有让她等太久。*

---

你想起身离开这个充满既视感的场景。樱花树后面似乎有一个小亭子，而湖边的小路还可以继续往前走。当然，你也可以回到礼堂去找小夏。

**你能感觉到，有些东西正在记忆深处蠢蠢欲动。**
