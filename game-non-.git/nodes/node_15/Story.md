# 故事：真相浮现

文件夹里的每一页都像是一把刀。

你翻动着那些纸张，手指在发抖。

**"创伤性记忆"**。**"选择性遗忘"**。**"申请者自愿"**。

每一个词都在告诉你同一件事——

你的失忆，不是意外。

是你自己选择的。

"大三那年。"小夏的声音从身后传来，像是在念一段尘封的日记。

"你家里出了一些事。我不知道具体是什么……你从来不肯告诉我。但是那天晚上……"

她的声音开始颤抖。

"那天晚上你来找我。浑身都在抖，脸色苍白得像纸。你抱着我，一直哭，一直说不想活了……"

你的心跳停了一拍。

"你说活着太累了。你说那些画面每天晚上都在梦里重演。你说你不要记得了。你说明明不是你的错，为什么你要记得……"

她说不下去了。

房间里只剩下她压抑的哭声。

"我求过你。"过了很久，她才继续说下去，声音嘶哑，"我求你不要忘记。我说我会陪着你，我说我们可以一起扛……"

"但是你说……"

她转过身，泪流满面地看着你。

"你说如果记得，你真的会死。"

---

台灯的光芒摇摇晃晃。

你看着手中的文件，看着那个签名。熟悉又陌生的笔迹，写下的却是你亲手选择的死刑判决书。

那不是字。

是你四年的记忆。

是你和小夏的一切。

---

**"如果记得，你真的会死。"**

**这是你给自己的最后通牒。**
